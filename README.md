# VAF HTL Grieskirchen Website

## Dependencies

- [hugo](https://gohugo.io/installation/)
- [pnpm](https://pnpm.io/)

## Run locally

- Run `pnpm install`
- Run `hugo server`

## Deploy

- Commit onto the `main` branch
- Push
- GitHub Actions will automatically deploy it
